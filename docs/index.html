💰 费用报销自动化工具

基于 OCR + AI 大模型，自动识别发票/收据/信用卡账单，智能分类、汇率换算、一键生成报销 Excel。

🇨🇳 中英文 OCR 🤖 DeepSeek AI 📊 Excel 输出 💱 自动汇率 🖥️ 本地运行
⬇ GitHub 下载 📖 使用指南

核心功能

📸

智能 OCR 识别

支持 PDF、JPG、PNG、HEIC 等格式。本地 RapidOCR 引擎，中英文混合识别，无需上传云端，隐私安全。

🤖

AI 智能分类

基于 DeepSeek 大语言模型，自动识别发票性质并归类。支持 14 种一级费用类目，根据文件名、商户、子文件夹智能判断。

💳

信用卡账单解析

支持美国信用卡 PDF 账单，可选「全部明细」或「仅高亮标记明细」两种模式。OpenCV 自动检测高亮区域。

💱

自动汇率换算

自动从中国人民银行外汇交易中心获取官方汇率，支持 USD / HKD / EUR / GBP / JPY 等 9 种货币。

👥

多人报销规则

每个报销人可配置独立的分类规则（餐饮→团建费 or 招待费），AI 会根据每人规则做个性化分类。

📁

文件自动重命名

处理后自动按「一级-二级-三级-币种-金额」格式重命名发票文件，整理归档省时省力。

🔍

智能去重

自动检测批次内重复和历史重复，避免同一张发票重复报销。模糊匹配，容忍 OCR 偏差。

📊

Excel 一键导出

按公司已有的报销表格格式自动生成 Excel。所有字段在导出前都可以在网页上编辑修改。

安装与使用

1克隆项目

git clone https://github.com/royhuroy/expensetool.git
cd expensetool

2安装依赖

pip install -r requirements.txt

3配置 API Key

复制 .env.example 为 .env，填入你的 DeepSeek API Key：

cp .env.example .env
# 编辑 .env 文件，填入：
DEEPSEEK_API_KEY=sk-你的密钥
💡 DeepSeek API Key 获取地址：platform.deepseek.com

4配置报销人员

编辑 data/persons.json，添加你公司的报销人员及分类规则：

{
  "persons": [
    {
      "name": "张三",
      "role": "投资同事",
      "dining_rule": "所有餐饮 → 招待费",
      "overseas_rule": "海外出差使用 美国-交通费 / 美国-招待费",
      "notes": "差旅较多"
    }
  ]
}

5放入发票 & 启动

将待报销的文件放入 invoices/ 文件夹（支持子文件夹），然后：

# Windows：双击「启动报销工具.bat」
# 或手动运行：
streamlit run src/app.py

浏览器将自动打开 http://localhost:8501，按界面提示操作即可。

6查看输出

处理完成后，output/ 文件夹中会生成：

  • expense_report_YYYY_MM.xlsx — 报销 Excel 报表
  • renamed_files/ — 按规则重命名的发票文件

支持的文件类型

📄 PDF 🖼️ JPG / JPEG 🖼️ PNG 📱 HEIC / HEIF 🖼️ TIFF

信用卡账单文件名包含「所有明细」→ 识别全部交易  |  包含「highlight」→ 仅识别高亮交易
